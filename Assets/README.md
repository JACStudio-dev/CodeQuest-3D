\# Arquitectura del Proyecto



Este proyecto utiliza una arquitectura modular.



\## Objetivos



\- Bajo acoplamiento.

\- Alta cohesión.

\- Separación clara de responsabilidades.

\- Escalabilidad.

\- Fácil mantenimiento.

\- Fácil incorporación de nuevos desarrolladores.



\---



\## Estructura



Core

Gameplay

Managers

Services

Data

UI

AI

Config

Settings

Persistence

Networking



Cada módulo posee su propio README.md donde se explica su responsabilidad.



No se deben colocar scripts fuera de su módulo correspondiente.
